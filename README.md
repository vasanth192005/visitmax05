# visitmax05
A React Native mobile application designed for marketing executives to track field visits, update clinic records, capture photos, and manage reports in real-time.

### **VisitMax – Healthcare Marketing & Field Visit Management Application**

**Project Description:**

**VisitMax** is a mobile application developed to streamline healthcare field marketing operations by enabling medical representatives and marketing executives to efficiently manage doctor visits, record daily activities, and generate real-time reports. The application improves productivity by digitizing the entire visit management process, reducing paperwork, and providing accurate tracking of field activities.

The system allows users to securely log in using OTP authentication, plan and record doctor or hospital visits, capture GPS location, upload visit proof through images, and maintain digital visit histories. Managers can monitor field performance through interactive dashboards, analytics, and comprehensive reports, enabling data-driven decision-making.

Built with a user-centered UI/UX approach, VisitMax focuses on simplicity, responsiveness, and ease of navigation, ensuring a seamless experience for field employees. The application supports role-based access control, secure API integration, and efficient database management to maintain reliable and scalable operations.

### **Key Features**

* OTP-based secure user authentication
* Role-based access (Admin, Manager, Marketing Executive)
* Doctor and hospital visit management
* GPS location tracking for field visits
* Image upload as visit proof
* Daily task and appointment scheduling
* Real-time dashboard and analytics
* Visit history and report generation
* Responsive mobile UI with intuitive navigation
* REST API integration with secure database connectivity

### **Technology Stack**

* **Frontend:** React Native
* **Backend:** Flask / FastAPI (Python)
* **Database:** MySQL / SQLite
* **Design Tool:** Figma
* **Tools:** Android Studio, Expo, Postman, Git
